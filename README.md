\# Work



このリポジトリは、業務用ファイルのテンプレートを管理するための保管場所です。



\## 保存するもの



\- draw.io のワークフローテンプレート

\- 共通で使う図のひな形

\- 作業手順やルールのサンプル

\- GitHub管理しても問題ない汎用ファイル



\## 保存しないもの



\- 実際の業務ワークフロー図

\- 製品名、品番、図番、取引先名を含むファイル

\- 社内の具体的な工程情報

\- 個人名や担当者名を含むファイル

\- 機密情報を含む資料



\## 運用ルール



1\. GitHubにはテンプレートだけ保存する。

2\. 実際に作成した業務用ファイルは、ローカルPCまたは社内NASに保存する。

3\. テンプレートを使うときは、直接編集せず、必ず名前を付けて別ファイルとして保存する。

4\. GitHubに保存する前に、社外に出して問題ない内容か確認する。
